# Resumen

En esta parte del trabajo se realiza un análisis preliminar de las variables involucradas en el set de datos de Cáncer de Mama de Wisconsin. Este consiste en una serie de características medidas para distintas muestras de células, y una variable a clasificar que determina si el tumor es maligno o benigno. 

# Objetivo

Se buscará explorar las variables, determinando la calidad del set de datos y encontrando relaciones entre ellas, y preparando las variables para un análisis de Machine Learning en una próxima entrega. 

# Contexto comercial

Un hospital de Wisconsin busca reducir gastos asociados a ensayos genéticos y patológicos de laboratorio para determinar si una biopsia de tejido mamario tiene características carcinógenas o no. Los geneticistas y patólogos son personal caro y necesitan mucha formación. Se nos elije como Data Scientist/Analyst para elaborar un modelo de ML que permita clasificar como Maligno o Benigno a una muestra de células basado únicamente en sus características morfológicas, cosas que un becario puede medir facilmente. De esta forma el hospital busca reducir sus costos en la detección y clasificación de muestras. 

# Hipótesis

Es posible generar un modelo de aprendizaje que utilice únicamente características morfológicas de una muestra de células para clasificarlas según si presentan características tumorales Malignas o Benignas
